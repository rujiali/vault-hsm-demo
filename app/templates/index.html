Retire old key versions — three-step demo

      Key Export — Control Group
      / Dual Approval Enforcement
    

    


      

        Why Control Groups?

        Even an authorised operator cannot export key material unilaterally.
        Vault's Control Group policy intercepts the request and issues a wrapping token — an opaque hold.
        A separate custodian must approve before the operator can unwrap and collect the key.
        This enforces four-eyes / dual-control without any application code.
